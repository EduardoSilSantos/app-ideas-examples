do repositório 
[app-ideas.](https://github.com/florinpop17/app-ideas)

## 📌 Descrição
O Bin2Dec é um aplicativo simples que converte números binários (compostos apenas por 0's e 1's) em números decimais.

## 🎯 Objetivos
Implementar uma aplicação que:
- [x] Permita ao usuário inserir um número binário válido
- [x] Valide a entrada para aceitar apenas 0 ou 1
- [x] Converta o número binário para decimal
- [x] Exiba o resultado de forma clara
