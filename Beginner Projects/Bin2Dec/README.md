# ✅ Challenge Completed: Bin2Dec App

This project implements a solution to the 
[Bin2Dec](https://github.com/florinpop17/app-ideas/blob/master/Projects/1-Beginner/Bin2Dec-App.md) 
challenge from the 
[app-ideas](https://github.com/florinpop17/app-ideas) repository.

## 📌 Description
Bin2Dec is a simple application that converts binary numbers (composed only of 0's and 1's) into decimal numbers.

## 🎯 Objectives
Implement an application that:
- [x] Allows the user to enter a valid binary number
- [x] Validates the input to accept only 0 or 1
- [x] Converts the binary number to decimal
- [x] Displays the result clearly
